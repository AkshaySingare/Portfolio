Product Recommendation System

I developed an E-Commerce application using React for the frontend and Node.js for the backend. The frontend and backend communicate using Axios, and MySQL is used as the database.

The key feature is a Product-to-Product Recommendation Engine that suggests relevant items to users based on their browsing patterns and interaction history.

Along with recommendations, the system includes several important features:

  • JWT Token Authentication for secure login.
  • Email Login & Email Notifications for order confirmed, shipped, and delivered updates.
  • Order History where users can view all previous and active orders.
  • Admin Panel to add, update, and delete products.
  • Order Status Management allowing admins to change status (Confirmed → Shipped → Delivered).
  • Analytics Dashboard showing graphs of most purchased products and total order trends.
